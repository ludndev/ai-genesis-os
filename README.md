# Genesis OS v0.0.1

This is a basic operating system project named "Genesis".

## Features

- 16-bit Real Mode Bootloader
- Loading Screen
- 32-bit Protected Mode Kernel
- Basic Video Memory Output

## Building and Running

### Prerequisites

- NASM
- GCC (MinGW or similar)
- LD
- Make
- QEMU

### Build

Run the following command to build the OS image:

```bash
make
```

### Run

Run the following command to launch QEMU:

```bash
make run
```
